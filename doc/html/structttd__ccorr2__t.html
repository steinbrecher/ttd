Detailed Description

Structure for coordinating second order cross-correlation calculation.

ttd_ccorr2_t stores the necessary information to calculate G(2) cross-correlations. Functions are provided for initialization, updating (adding a new time record), writing the result to a CSV file, and cleaning up memory allocations. Using the member parameter window_time, the update function handles buffering of previously seen time records.

The output format is a histogram, with binned coincidences (ttd_ccorr2_t::bin_time sets the size of these), which is written to a csv file, and can be normalized for g(2) correlations.
